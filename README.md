# OSsMap
===========

Projeto de conclusão de curso de pós-graduação em Desenvolvimento Web pelo IFSul de Minas - Campus Passos.

## O projeto
===========

A disponibilização e utilização da informação no momento correto dentro de um processo operacional é um verdadeiro resumo do advento tecnológico que se consolida nos tempos atuais. O uso de equipamentos portáteis e conectados é uma experiência de software ímpar onde pode-se disponibilizar dados para a melhor execução de uma tarefa. Uma aplicação que gerencie serviços prestados por empresas de saneamento básico aliados ao Sistema de Informações Geográficas (SIG) pode tornar-se essencial para a execução do serviço.
Este projeto trata do desenvolvimento de uma solução computacional na problemática de empresas de saneamento básico, no que se entende por serviços prestados em campo, para suprir a falta de informação de estruturas básicas, como redes de distribuição e recolhimento de dejetos, hidrômetros e demais estruturas já instaladas, e o motivo da realização do serviço, o que pode acarretar impedimentos ou prejuízos de ordem patrimonial ou financeira.
Aliando a solução ao SIG é possível proporcionar às equipes de trabalho informações pertinentes ao serviço que será realizado por meio de equipamentos portáteis com aceso à Internet, que em muito podem contribuir para o sucesso da operação.
